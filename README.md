# 👟 One8 Sneakers — Rule The Pitch. Own The Drive.

A premium, modern landing page designed to showcase **One8 Sneakers**, the elite cricket-inspired footwear collection co-created with **Virat Kohli**. This single-page application is built using semantic HTML5, pure CSS3 (featuring custom layouts, fluid transitions, and glassmorphism), and interactive vanilla JavaScript.

---

## 🎨 Design & Key Features

*   **Premium Visual Aesthetics:** Sleek dark/light contrast theme utilizing a carefully curated color palette of Turquoise, Gold, and Charcoal.
*   **Dynamic Interactive Components:**
    *   **Galaxy Glow CTA:** Hover animations with a custom conic-gradient border and floating stars.
    *   **Collection Showcase:** Interactive product grid displaying four unique silhouettes with a dynamic float effect and image-swap hover states.
    *   **Performance Metrics:** Highlights the engineering detail behind the footwear with smooth hover states.
*   **Fully Responsive & Accessible:** Optimized for seamless performance on desktop, tablet, and mobile devices with a custom responsive navigation drawer.
*   **Smooth Navigation:** Built-in CSS scroll behaviors mapping to all major page sections.

---

## 🛠️ Technology Stack

*   **Structure:** HTML5 (Semantic tags, full responsiveness)
*   **Styling:** Custom Vanilla CSS3 (CSS Variables, Flexbox, CSS Grid, Custom Keyframe Animations)
*   **Interactivity:** Vanilla JavaScript (Dynamic header shadow, mobile menu controls, interactive alerts)
*   **Brand Assets:** Scalable Vector Graphics (SVG)
